# ScriptPal 🤘

A simple npm script palette for lazy people (like me), who want a quick way to look through and pick npm scripts!

- keyboard navigation
- autocompletion
- fussy finding

## Install ⬇️

Install globally

```bash
npm install -g scriptpal
```

## Get started 🏁

```bash
scriptpal
```

Usage with npx

```bash
npx scriptpal
```
